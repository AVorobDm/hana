#define BOOST_HANA_CONSTEXPR_LAMBDA   constexpr

Expands to constexpr if constexpr lambdas are supported and to nothing otherwise.

#define BOOST_HANA_STATIC_ASSERT (   ...)    assert((__VA_ARGS__))

Expands to a runtime assertion.

Use this only at function scope. This is a workaround because constexpr lambdas are not available yet and so our unit tests can't use static_assert.
