boost::hana::Product Struct Reference
Type classes

Description

Represents types that are generic containers of two elements.

Specifically, the Product type class represents types that are category theoretical products.

Laws

For an instance P of Product, the following laws must be satisfied, which is equivalent to satisfying the universal property exposed in the article linked above. For every data types X, P1, P2 and functions f : X -> P1, g : X -> P2, there exists a unique function make : X -> P such that for all object x of data type X,

f(x) == first(make(x))
g(x) == second(make(x))

This is basically saying that a Product must be the most general object able to contain a pair of objects (P1, P2), but nothing more. Since the categorical product is defined by a universal property, all instances are unique up to isomorphism.
