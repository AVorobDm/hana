boost::hana::Functor Struct Reference
Type classes

Description

Functor represents types that can be mapped over.

Instances of Functor must satisfy the following laws. For any Functor x and functions with matching domains/codomains f and g,

fmap(id, x) == x
fmap(compose(f, g), x) == fmap(f, fmap(g, x))

Map f over a Functor.

Parameters
fA function called as f(x) on each element x of the structure, and returning a new value to replace x in the structure.
functorThe structure to map f over.

Example

auto to_string = [](auto x) {
return (std::ostringstream{} << x).str();
};
assert(
fmap(to_string, list(1, '2', "345", std::string{"67"}))
==
list("1", "2", "345", "67")
);
BOOST_HANA_STATIC_ASSERT(fmap(to_string, nothing) == nothing);
assert(fmap(to_string, just(123)) == just("123"));

Example

BOOST_HANA_STATIC_ASSERT(
fmap(template_<std::add_pointer_t>, type_list<void, int(), char[10]>)
==
type_list<void*, int(*)(), char(*)[10]>
);

Apply a function on all the elements of a structure satisfying a predicate.

Parameters
predicateA function called as predicate(x) for each element x of the structure and returning a Logical representing whether f should be applied to x.
fA function called as f(x) on each element for which the predicate returns a true-valued Logical.
functorThe structure to map f over.

Example

BOOST_HANA_CONSTEXPR_LAMBDA auto negative = [](auto x) {
return x < 0;
};
BOOST_HANA_CONSTEXPR_LAMBDA auto negate = [](auto x) {
return -x;
};
BOOST_HANA_STATIC_ASSERT(
adjust(negative, negate, list(-3, -2, -1, 0, 1, 2, 3))
==
list(3, 2, 1, 0, 1, 2, 3)
);

Replace all the elements of a structure with a fixed value.

Parameters
