boost::hana::Map Struct Reference
Data types

Description

A basic associative array requiring unique and Comparable keys.

The order of the elements of the map is unspecified. Also, all the keys must be comparable with each other and that comparison must yield a compile-time Logical.

Modeled concepts

  1. Comparable (operators provided)
    Two maps are equal iff all their keys are equal and are associated to equal values.
    using namespace std::literals;
    BOOST_HANA_RUNTIME_CHECK(
    make_map(
    pair(char_<'a'>, "foobar"s),
    pair(type<int&&>, nullptr)
    )
    ==
    make_map(
    pair(type<int&&>, (void*)0),
    pair(char_<'a'>, "foobar"s)
    )
    );
  2. Searchable
    A map can be searched by its keys with a predicate yielding a compile-time Logical.
    BOOST_HANA_CONSTEXPR_LAMBDA auto m = make_map(
    pair(type<int>, 'i'),
    pair(type<float>, 'f')
    );
    BOOST_HANA_CONSTEXPR_CHECK(find(m, type<int>) == just('i'));
    BOOST_HANA_CONSTEXPR_CHECK(find(m, type<float>) == just('f'));
    BOOST_HANA_CONSTANT_CHECK(find(m, type<void>) == nothing);
    BOOST_HANA_CONSTANT_CHECK(find(m, int_<3>) == nothing);

Provided conversions

  1. From any Record
    Converting a Record R to a Map is equivalent to converting its members<R>() to a Map, except the values are replaced by the actual members of the object instead of accessors.
  2. From any Foldable
    Converts a Foldable of Products to a Map. Note that the foldable structure must not contain duplicate keys.
    Todo:
    We should allow duplicate keys, with a documented policy (e.g. we keep the last one).
  3. To any Sequence
    A Map can be converted to a Sequence of Products.

Create a Map with the given key/value associations.

Given zero Products or more representing key/value associations, make<Map> returns a Map associating these keys to these values. All the keys must be unique.

Example

using namespace std::literals;
auto m = make<Map>(
make<Pair>(int_<1>, "foobar"s),
make<Pair>(type<void>, 1234)
);
constexpr auto make_map = make<Map>
