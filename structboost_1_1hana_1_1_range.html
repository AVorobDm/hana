Creates a Range containing the Integrals in [from, to).

from and to must be Integrals such thatÂ from <= to. Otherwise, a compilation error is triggered.

Example

BOOST_HANA_CONSTANT_ASSERT(head(range(0_c, 5_c)) == 0_c);
BOOST_HANA_CONSTANT_ASSERT(last(range(0_c, 5_c)) == 4_c);

Shorthand to create a Range of Integrals.

This shorthand is provided for convenience only and it is equivalent to using range. Specifically, range_c<T, from, to> is such that

range_c<T, from, to> == range(integral<T, from>, integral<T, to>)
Template Parameters
TThe underlying integral type of the Integrals in the created range.
fromThe inclusive lower bound of the created range.
toThe exclusive upper bound of the created range.

Example

BOOST_HANA_CONSTANT_ASSERT(head(range_c<int, 0, 5>) == int_<0>);
BOOST_HANA_CONSTANT_ASSERT(last(range_c<unsigned long, 0, 5>) == ulong<4>);
BOOST_HANA_CONSTANT_ASSERT(tail(range_c<int, 0, 5>) == range(int_<1>, int_<5>));
