boost::hana::Monad::instance< Maybe > Struct Template Reference

template<>
struct boost::hana::Monad::instance< Maybe >

Instance of Monad for Maybes.

The Maybe Monad makes it easy to compose actions that might fail. One can feed an optional value if there is one into a function with bind, which will return nothing if there is no value. Finally, optional-optional values can have their redundant level of Maybeness removed with flatten.

Example

/*

(See accompanying file LICENSE.md or copy at http://boost.org/LICENSE_1_0.txt)

#include <boost/hana/detail/constexpr.hpp>
#include <boost/hana/detail/static_assert.hpp>
#include <boost/hana/maybe.hpp>
using namespace boost::hana;
int main() {
BOOST_HANA_CONSTEXPR_LAMBDA auto inc = [](auto x) {
return just(x + 1);
};
BOOST_HANA_STATIC_ASSERT(bind(just(1), inc) == just(2));
BOOST_HANA_STATIC_ASSERT(bind(nothing, inc) == nothing);
BOOST_HANA_STATIC_ASSERT(flatten(just(just(2))) == just(2));
}
