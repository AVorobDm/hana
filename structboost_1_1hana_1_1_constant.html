boost::hana::Constant Struct Reference
Type classes

Description

The Constant type class is used for data types representing compile-time constants.

Laws

For any Constant c, the following program must be well-formed:

template <typename X>
void f(X x) {
constexpr auto y = value(x);
}
int main() {
f(c);
}

This is basically saying that all of c's information used in the call to value must be stored in its type.

Return the compile-time value of a constant.

This function returns a value which is always a constant expression, and this function can always be called inside a constant expression. This poses some rather heavy restrictions on the implementation; specifically, it means that the implementation is fully determined by the type of its argument, and that it does not use the value of its argument at all.

Example

auto i = integral<int, 3>; // notice no constexpr
static_assert(value(i) == 3, "value(i) is always a constant expression!");
