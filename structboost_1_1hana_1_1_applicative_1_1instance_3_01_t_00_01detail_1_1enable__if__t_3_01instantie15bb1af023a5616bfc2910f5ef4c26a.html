Detailed Description

template<typename T>
struct boost::hana::Applicative::instance< T, detail::enable_if_t< instantiates< List, T > > >

A value can be lifted into a singleton list with lift. ap(fs, xs) applies each function in the list fs to each value in the list xs, and returns a list containing all the results.

Example

BOOST_HANA_STATIC_ASSERT(lift<List>('a') == list('a'));
BOOST_HANA_CONSTEXPR_LAMBDA auto f = curry<2>(_ + _);
BOOST_HANA_STATIC_ASSERT(
ap(ap(lift<List>(f), list(1, 2)), list(3, 4, 5)) ==
list(
f(1, 3), f(1, 4), f(1, 5),
f(2, 3), f(2, 4), f(2, 5)
)
);
