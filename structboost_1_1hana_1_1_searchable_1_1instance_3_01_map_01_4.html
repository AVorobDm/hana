boost::hana::Searchable::instance< Map > Struct Template Reference

Description

template<>
struct boost::hana::Searchable::instance< Map >

A map can be searched by its keys with a predicate yielding a boolean Integral.

Example

auto m = map(
pair(type<int>, 'i'),
pair(type<float>, 'f')
);
BOOST_HANA_CONSTEXPR_ASSERT(lookup(type<int>, m) == just('i'));
BOOST_HANA_CONSTEXPR_ASSERT(lookup(type<float>, m) == just('f'));
BOOST_HANA_CONSTANT_ASSERT(lookup(type<void>, m) == nothing);
BOOST_HANA_CONSTANT_ASSERT(lookup(int_<3>, m) == nothing);
