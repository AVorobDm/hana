boost::hana::Maybe Struct Reference
Data types

Description

Represents an optional value.

A Maybe either contains a value (represented as just(x)), or it is empty (represented as nothing).

Instance of

Comparable, Functor, Applicative, Monad, Foldable and Searchable

Create a Maybe with the result of a function, but only if a predicate is satisfied.

Specifically, returns just(f(x)) if predicate(x) is a true-valued Logical, and nothing otherwise.

Parameters
predicateA function called as predicate(x) and returning a true-valued Logical if just(f(x)) should be the resulting value, and a false-valued one if nothing should be the resulting value. In the current version of the library, the result of predicate has to be a compile-time Logical.
fA function called as f(x) if the predicate returns a true-valued Logical, and not called at all otherwise. If the predicate returns a false-valued Logical, the f(x) expression is not even required to be well-formed.
xThe value to either transform and put in a just, or discard.

Example

BOOST_HANA_CONSTEXPR_LAMBDA auto even = [](auto x) {
return x % int_<2> == int_<0>;
};
BOOST_HANA_CONSTEXPR_LAMBDA auto half = [](auto x) {
return x / int_<2>;
};
BOOST_HANA_STATIC_ASSERT(only_when(even, half, int_<4>) == just(int_<2>));
BOOST_HANA_STATIC_ASSERT(only_when(even, half, int_<3>) == nothing);

Apply a function to the contents of a Maybe, with a fallback result.

Specifically, maybe takes a default value, a function and an optional value. If the optional value is nothing, the default value is returned. Otherwise, the function is applied to the content of the just.

Parameters
default_A default value returned if m is nothing.
fA function called as f(x) if and only if m is an optional value of the form just(x). In that case, the result returend by maybe is the result of f.
mAn optional value.

Example

BOOST_HANA_CONSTEXPR_LAMBDA auto inc = [](auto x) { return x + 1; };
BOOST_HANA_STATIC_ASSERT(maybe('x', inc, just(1)) == 2);
BOOST_HANA_STATIC_ASSERT(maybe('x', inc, nothing) == 'x');

Return whether a Maybe contains a value.

Specifically, returns a compile-time true-valued Logical if m is of the form just(x) for some x, and a false-valued one otherwise.

Example

BOOST_HANA_STATIC_ASSERT( is_just(just('x')));
BOOST_HANA_STATIC_ASSERT( is_just(just(nothing)));
BOOST_HANA_STATIC_ASSERT(!is_just(nothing));

Specifically, returns a compile-time true-valued Logical if m is of the form nothing, and a false-valued one otherwise.

Return the contents of a Maybe, with a fallback result.

Specifically, returns x if m of the form just(x), and default_ if m is of the form nothing.

Parameters
default_The default value to return if m is nothing.
mThe optional value to try to retrieve the value from.

Example

BOOST_HANA_STATIC_ASSERT(from_maybe('x', just(1)) == 1);
BOOST_HANA_STATIC_ASSERT(from_maybe('x', nothing) == 'x');

Extract the content of a Maybe or fail at compile-time.

Specifically, returns x if the optional value is of the form just(x), and triggers a static assertion otherwise.

Example

BOOST_HANA_STATIC_ASSERT(from_just(just('x')) == 'x');
// from_just(nothing); // compile-time static assertion
