Definition: string.hpp:24

Create a compile-time string from a string literal s.

This macro is the only way to create a compile-time string. Also, since this macro uses a lambda internally, it can't be used in an unevaluated context.

Example

BOOST_HANA_CONSTEXPR_LAMBDA auto s = BOOST_HANA_STRING("abcdef");
