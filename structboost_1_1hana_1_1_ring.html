boost::hana::Ring Struct Reference
Type classes

Description

The Ring type class is used for Groups that also form a Monoid under a second binary operation that distributes over the first.

The method names refer to the ring of numbers under multiplication.

Requires

Group

Laws

For all objects x, y, z of a Ring R, the following laws must be satisfied:

mult(x, mult(y, z)) == mult(mult(x, y), z) // associativity
mult(x, one<R>) == x // right identity
mult(one<R>, x) == x // left identity
mult(x, plus(y, z)) == plus(mult(x, y), mult(x, z)) // distributivity
